# Short Presentation Script

Hello everyone, today I will present a simple Java console demo.

1. **OOP design**
   - `Student` is the model class.
   - Fields are private and accessed through getters/setters.
   - Validation is done in constructors/setters to keep data clean.

2. **CRUD operations**
   - `StudentService` stores students in an `ArrayList`.
   - It supports create, read, update, and delete by student id.

3. **Stream API**
   - `filter`: select students by minimum age.
   - `sorted`: sort students by name.
   - `map` + `forEach`: transform names to uppercase and print.

4. **Loop comparison**
   - I also show a traditional `for` loop for the same filter logic.
   - This helps compare readability between classic Java and Stream API.

In short, this project demonstrates core Java skills in a small, easy-to-understand example that is good for presentations.
